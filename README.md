## English Version

### Overview
Steve Jobs Thinking Advisor is a comprehensive AI assistant based on Steve Jobs's core philosophies, designed to help users apply Jobs's thinking patterns to product design, innovation strategy, team management, and entrepreneurship challenges.

### Key Features
- **Product Design Analysis**: Apply Jobs's simplicity philosophy to evaluate and improve product designs
- **Innovation Strategy**: Use "Think Different" framework to break industry conventions
- **Team Management**: Build pirate-style innovation teams with A-player culture
- **Leadership Development**: Learn Jobs's reality distortion field and high-standard leadership
- **Case Learning**: Study 6 success cases and 4 failure cases with detailed analysis
- **Thinking Tools**: 5 practical templates for applying Jobs's methods
- **Knowledge Graph**: Structured data with 20 entities and 20 relationships

### Project Structure
```
jobs-advisor/
├── jobs-advisor-en/          # English version
│   ├── SKILL.md              # Core skill definition with conversation examples
│   ├── cases.md              # Success and failure case analysis
│   ├── tools.md              # Thinking tool templates
│   └── README.md             # English documentation
├── jobs-advisor-zh/          # Chinese version
│   ├── SKILL.md              # 核心技能定义（含对话示例）
│   ├── cases.md              # 成功与失败案例分析
│   ├── tools.md              # 思维工具模板
│   └── README.md             # 中文文档
├── knowledge-graph/          # Knowledge graph data
│   ├── entities.json         # 20 core entities
│   ├── relationships.json    # 20 core relationships
│   ├── simple_graph.json     # Simplified graph for visualization
│   └── graph_visualization.png # Graph visualization image
├── jobs_research_summary.md  # Complete research summary
└── README.md                 # This file (bilingual)
```

### Quick Start
1. **For AI Assistants**: Load the skill from `jobs-advisor-en/SKILL.md` or `jobs-advisor-zh/SKILL.md`
2. **For Learning**: Read `cases.md` for real-world examples and `tools.md` for practical templates
3. **For Analysis**: Use the knowledge graph data in `knowledge-graph/` for structured analysis

### Core Thinking Framework
1. **Simplicity Philosophy**: Less is more, focus on what matters
2. **Reality Distortion Field**: Make impossible possible
3. **Pirate Spirit**: Small teams beat big companies
4. **Think Different**: Challenge conventions, create new categories
5. **End-to-End Control**: Hardware + software + services integration

## 中文版本

### 概述
史蒂夫·乔布斯思维助手是基于乔布斯核心思想的AI助手，帮助用户应用乔布斯的产品哲学、创新思维、设计理念和领导风格分析产品设计、创新策略、团队管理和创业问题。

### 主要功能
- **产品设计分析**：应用简洁哲学评估和改进产品设计
- **创新策略制定**：使用"不同凡想"框架突破行业常规
- **团队管理指导**：建立海盗式创新团队和A级人才文化
- **领导力发展**：学习现实扭曲力场和高标准领导
- **案例学习**：研究6个成功案例和4个失败案例
- **思维工具**：5个实用模板应用乔布斯方法
- **知识图谱**：结构化数据，包含20个实体和20个关系

1. **简洁哲学**：少即是多，专注核心

3. **海盗精神**：小团队打败大公司
4. **不同凡想**：挑战常规，创造新品类
5. **端到端控制**：硬件+软件+服务一体化
